
# ⚡ Simulador de Campo Eléctrico (Disco Cargado)

Simulación en Python del campo eléctrico generado por un disco con carga uniforme y la fuerza sobre una partícula.

## ¿Que hace?
- Calcula el campo eléctrico en el eje del disco
- Calcula la fuerza sobre una partícula
- Visualiza:
  - Disco cargado
  - Partícula
  - Vector campo eléctrico
  - Vector fuerza

## Tecnologías
- Python
- NumPy
- Matplotlib
